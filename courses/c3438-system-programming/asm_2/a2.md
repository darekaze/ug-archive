## 1. Regex

```tex
KEYWORD   -> var|begin|end
COMMA     -> ,
SEMICOLON -> ;
ASSIGN    -> =
PERIOD    -> \.
NUM       -> [0-9]*(\.([0-9]+))?
PLUS      -> +
MINUS     -> -
MUL       -> *
DIV       -> /
LBRACE    -> (
RBRACE    -> )
ID        -> [a-zA-Z][a-zA-Z0-9]*
```
